# 🏎️ F1 Dashboard Interattiva

Dashboard dinamica e completa per l'analisi dei dati Formula 1 degli ultimi 3 anni (2023-2026).

## ✨ Caratteristiche

- 📊 Grafici interattivi con Plotly
- 🔍 Filtri avanzati per anno, gara, pilota e sessione
- 🏆 Classifiche mondiali piloti e costruttori
- ⏱️ Analisi tempi sul giro e telemetria
- 📍 Evoluzione posizioni in tempo reale
- 🎨 Design moderno e responsive

## 🚀 Installazione

# Clona il repository
git clone https://github.com/tuo-username/f1-dashboard.git
cd f1-dashboard

# Installa le dipendenze
pip install -r requirements.txt

# Avvia l'applicazione
streamlit run app.py

## 📦 Dipendenze

- Streamlit 1.31.0
- Pandas 2.2.0
- Plotly 5.18.0
- Requests 2.31.0
- FastF1 3.3.0 (opzionale per telemetria avanzata)

## 🔑 API

Utilizza [OpenF1 API](https://openf1.org) per dati storici gratuiti dal 2023.

## 📖 Utilizzo

1. Seleziona l'anno dalla sidebar
2. Scegli la gara desiderata
3. Seleziona il tipo di sessione (Gara, Qualifiche, Prove)
4. Esplora i grafici e le statistiche
5. Filtra per piloti specifici

## 🎯 Funzionalità Avanzate

- **Tempi sul Giro**: Confronta fino a 10 piloti contemporaneamente
- **Evoluzione Posizioni**: Visualizza sorpassi e strategie
- **Pit Stop**: Analizza strategie di sosta
- **Degrado Gomme**: Valuta prestazioni per stint
- **Race Control**: Messaggi safety car e bandiere
